# cheongwon-data-visualization
청원24 데이터 수집 및 시각화
* 크롤링으로 청원24 데이터 수집
* 수집한 데이터 엑셀에 저장
* 엑셀에 저장한 데이터 불러오기
* KoLNPy(자연어 처리 패키지)를 이용해서 데이터에서 단어 추출
* 빈도수가 가장 높은 단어 추출
* 추출한 단어를 WordCloud로 시각화
